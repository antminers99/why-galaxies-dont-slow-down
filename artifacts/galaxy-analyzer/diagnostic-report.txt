SINGLE DEFENSIBLE CLAIM
================================================================================

  "There is a real acceleration transition scale in galaxy rotation
  curves. After sample cleaning, per-galaxy Upsilon_star marginalization,
  and hierarchical modeling on the full (not subsampled) dataset, our
  most defensible estimate is
  a₀ = 3633 (km/s)²/kpc = 1.18×10⁻¹⁰ m/s², with substantial
  between-galaxy heterogeneity (tau = 0.291 dex, I²=92.4%) and total
  systematic uncertainty ±0.32 dex. The literature value
  (1.20×10⁻¹⁰ m/s²) falls within our 1σ band.
  The cosmological ratio a₀/(cH₀/2π) = 1.13 is suggestive but shows
  moderate variation across subsamples (CV = 14.1%)."

  This claim is narrower than anything in the previous report.
  It is also the only claim that survives all diagnostic tests below.

  What we do NOT claim: a₀ confirmed as universal exact constant,
  dark matter solved, MOND proved, or cosmological origin established.

SECTION 2: INTERPOLATION FUNCTION AUDIT (Test 1)
================================================================================

  PURPOSE: Does the fitted a₀ depend on the interpolation function?
  METHOD: Fit a₀ by minimizing RMS(log g_obs - log g_pred) using
          golden section search over log(a₀) ∈ [2.0, 5.0].
  DATA: 2,062 points from 193 galaxies.

  ┌──────────────────────────────────┬──────────┬────────┬─────────┬─────────┐
  │ Interpolation Function           │ a₀       │ RMS    │ AIC     │ BIC     │
  │                                  │(km/s)²/kpc│ (dex) │         │         │
  ├──────────────────────────────────┼──────────┼────────┼─────────┼─────────┤
  │ McGaugh RAR: 1/(1-e^(-√y))      │ 1809     │ 0.2385 │ -5909.8 │ -5904.1 │
  │ Simple MOND: 1/2 + √(1/4+1/y)   │ 1988     │ 0.2385 │ -5909.7 │ -5904.1 │
  │ Standard MOND: (1+√(1+4/y))/2   │ 1988     │ 0.2385 │ -5909.7 │ -5904.1 │
  │ Our ν(x)=x/(1+x)                │ 1972     │ 0.2443 │ -5810.8 │ -5805.2 │
  └──────────────────────────────────┴──────────┴────────┴─────────┴─────────┘

  KEY FINDINGS:

  1. a₀ varies by only 4.1% across 4 interpolation functions.
     The transition scale is ROBUST to model choice.

  2. All functions give a₀ ≈ 1900 (km/s)²/kpc in the global fit.
     This is LOWER than the canonical 3702 from SPARC per-galaxy fits.
     Reason: global fit weights all points equally; per-galaxy median
     weights all galaxies equally. Low-mass galaxies dominate by number
     but have fewer well-constrained points.

  3. Our framework (ν(x)=x/(1+x)) fits WORST of the 4.
     ΔAIC = 99 relative to McGaugh RAR form — statistically decisive.
     The McGaugh and standard MOND forms are indistinguishable.

  4. The cosmological ratio a₀/(cH₀/2π) is 0.56–0.62 with the global
     fit, NOT ~0.87 as previously reported. The ~0.87 ratio uses the
     per-galaxy median a₀ = 3702, which is method-dependent.

  IMPLICATIONS:
  - The EXISTENCE of a transition scale is model-independent.
  - The PRECISE VALUE of a₀ is model-dependent (global vs per-galaxy,
    interpolation function, sample composition).
  - The cosmological coincidence a₀ ≈ cH₀/2π depends on methodology.
  - Our interpolation function is not the best fit to the data.

SECTION 3: BARYONIC CALIBRATION FACTOR (Test 2)
================================================================================

  PURPOSE: Does rescaling g_bar by a constant factor improve the fit?
  METHOD: Replace g_bar → f_bar × g_bar, fit with McGaugh RAR and
          fixed a₀ = 3702, scan f_bar from 0.7 to 1.5.

  ┌─────────┬──────────┬──────────┐
  │ f_bar   │ RMS (dex)│ bias     │
  ├─────────┼──────────┼──────────┤
  │ 0.70    │ 0.2436   │ +0.030   │
  │ 0.80    │ 0.2425   │ -0.005   │
  │ 0.90    │ 0.2416   │ -0.036   │
  │ 0.95    │ 0.2412   │ -0.051   │
  │ 1.00    │ 0.2409   │ -0.065   │
  │ 1.10    │ 0.2403   │ -0.090   │
  │ 1.20    │ 0.2399   │ -0.114   │
  │ 1.30    │ 0.2395   │ -0.136   │
  │ 1.50    │ 0.2390   │ -0.175   │
  └─────────┴──────────┴──────────┘

  KEY FINDINGS:

  1. RMS is nearly FLAT across all f_bar values.
     Best f_bar = 1.64, but improvement is only 0.9%.
     Baryonic calibration is NOT the dominant source of scatter.

  2. Bias changes monotonically with f_bar.
     f_bar = 0.80 gives near-zero bias (-0.005 dex).
     This suggests Υ_★ may be slightly overestimated, OR
     a₀ = 3702 is too high for the global fit (consistent with Test 1).

  3. f_bar and a₀ are DEGENERATE.
     Scaling g_bar up is equivalent to scaling a₀ down.
     To separate them: need external Υ_★ constraint
     (e.g., stellar population synthesis, dynamical mass).

  IMPLICATIONS:
  - Baryonic calibration is not the "hidden factor."
  - The scatter is dominated by galaxy-to-galaxy variation, not Υ_★ offset.
  - A ~20% Υ_★ offset cannot be ruled out but does not change conclusions.

SECTION 6: PARTIAL CORRELATIONS (Test 5 — Phase 2)
================================================================================

  PURPOSE: The 4 galaxy properties (gas fraction, surface brightness,
           luminosity, V_max) are correlated with each other. Which one
           is the TRUE driver of residual structure?
  METHOD: Partial correlations (controlling for confounds) and
          multivariate regression on 157 galaxies.

  SIMPLE (zero-order) correlations with RAR residual:
  ┌─────────────────────────┬──────────┬──────────────────────┐
  │ Variable                │ r        │ Assessment           │
  ├─────────────────────────┼──────────┼──────────────────────┤
  │ V_max                   │ +0.414   │ SIGNIFICANT ⚠        │
  │ Luminosity (L36)        │ +0.158   │ Weak                 │
  │ Surface brightness      │ +0.084   │ Not significant      │
  │ Gas fraction (MHI/L36)  │ -0.079   │ Not significant      │
  └─────────────────────────┴──────────┴──────────────────────┘

  PARTIAL correlations (controlling for 1 variable):
  ┌──────────────────────────────────────────┬──────────┐
  │ Partial correlation                      │ r        │
  ├──────────────────────────────────────────┼──────────┤
  │ r(resid, Vmax | SB)                     │ +0.610   │
  │ r(resid, Vmax | gasFrac)                │ +0.546   │
  │ r(resid, gasFrac | Vmax)                │ +0.397   │
  │ r(resid, SB | Vmax)                     │ -0.498   │
  │ r(resid, L | SB)                        │ +0.173   │
  │ r(resid, L | gasFrac)                   │ +0.170   │
  │ r(resid, gasFrac | L)                   │ +0.100   │
  │ r(resid, SB | L)                        │ -0.109   │
  │ r(resid, SB | gasFrac)                  │ +0.033   │
  │ r(resid, gasFrac | SB)                  │ -0.017   │
  └──────────────────────────────────────────┴──────────┘

  KEY FINDINGS:

  1. V_max is the PRIMARY residual driver. It STRENGTHENS when
     controlling for SB (r = +0.414 → +0.610) or gas fraction
     (r → +0.546). This means V_max is the fundamental variable,
     partly suppressed by confounders.

  2. SB becomes significant only when controlling for Vmax
     (r = +0.084 → -0.498). This is a suppression effect:
     SB and Vmax are correlated, and their effects partially
     cancel in the simple correlation.

  3. Gas fraction and SB are NOT independently significant
     (r(gas | SB) = -0.017, r(SB | gas) = +0.033).
     They carry the same information.

  4. Luminosity adds nothing beyond SB and gas fraction
     (r(L | SB) = +0.173, r(L | gas) = +0.170).

  HIERARCHY: Vmax >> SB (conditional on Vmax) >> gas fraction ≈ L

SECTION 7: MULTIVARIATE RESIDUAL MODEL (Test 6 — Phase 2)
================================================================================

  PURPOSE: How much residual variance can galaxy properties explain?
  METHOD: Ordinary least squares regression.
           resid = b₀ + b₁×logGasFrac + b₂×logSB + b₃×logVmax + b₄×logL

  MODEL HIERARCHY:
  ┌──────────────────────────────────────┬────────┬────────┬────────┐
  │ Model                                │ R²     │ adj-R² │ ΔR²   │
  ├──────────────────────────────────────┼────────┼────────┼────────┤
  │ gasFrac only                         │ 0.006  │ -0.007 │ —      │
  │ SB only                              │ 0.007  │ -0.006 │ —      │
  │ gasFrac + SB                         │ 0.007  │ -0.012 │ 0.000  │
  │ gasFrac + SB + Vmax                  │ 0.394  │ 0.378  │ +0.387 │
  │ gasFrac + SB + Vmax + L (full)      │ 0.638  │ 0.626  │ +0.244 │
  └──────────────────────────────────────┴────────┴────────┴────────┘

  FULL MODEL COEFFICIENTS (n=157):
    intercept      = -2.205
    logGasFrac     = -0.050
    logSB          = -0.108
    logVmax        = +1.561  (DOMINANT)
    logL           = -0.302

  KEY FINDINGS:

  1. Gas fraction and SB ALONE explain essentially NOTHING
     (R² = 0.006 and 0.007). This confirms that the Phase 1
     "strong correlations" were artifacts.

  2. Adding Vmax jumps R² from 0.007 to 0.394. Vmax is the
     dominant predictor by a wide margin.

  3. Adding L on top of Vmax raises R² to 0.638. Together,
     Vmax and L explain 64% of residual variance. But this
     is NOT alarming — it reflects the mass-dependent offset
     from using a₀ = 3702 instead of the fitted a₀ = 2670.

  4. SPARC-only model: R² = 0.643 with identical coefficients.
     The pattern is not driven by LITTLE THINGS contamination.

SECTION 11: METHODOLOGY NOTES
================================================================================

  DATA FORMAT:
    Plot points stored as (x, y, g) where:
      x = log10(g_bar) in (km/s)²/kpc
      y = log10(g_obs/g_bar)
      g = galaxy name (truncated to 12 chars)

  g_bar COMPUTATION:
    From SPARC: Vbar² = Υ_disk × Vdisk² + Υ_bulge × Vbulge² + Vgas²
    g_bar = Vbar² / r
    Υ_disk = 0.5, Υ_bulge = 0.7 (fixed, not fitted per galaxy)
    This is a proper disk mass model, NOT spherical GM/r².

  LITTLE THINGS:
    g_bar derived from V_total² - V_DM² (using published DM models)
    Less reliable than SPARC decomposition.
    Only 2 LITTLE THINGS galaxies in analysis subset (most lack metadata).

  FITTING:
    Golden section search on log(a₀) with 50 refinement steps.
    Cost function: RMS of log10(g_obs) - log10(g_pred).
    No marginalization over nuisance parameters.
    1 free parameter (a₀) per fit.

  INTERPOLATION FUNCTIONS TESTED:
    1. McGaugh RAR: g_obs = g_bar / (1 - exp(-√(g_bar/a₀)))
    2. Simple MOND: g_obs = g_bar × (1/2 + √(1/4 + a₀/g_bar))
    3. Standard MOND: g_obs = g_bar × (1 + √(1 + 4a₀/g_bar)) / 2
    4. Our framework: g_obs = √(g_bar² + a₀² × g_bar/(g_bar + a₀))

  RESIDUAL COMPUTATION (corrected in Phase 2):
    Per-galaxy mean residual:
      gbar = 10^(p.x)                         [correct]
      gobs = gbar × 10^(p.y)
      pred = mcgaughRAR(gbar, a₀)
      resid = log10(gobs) - log10(pred)
    Phase 1 BUG: used gbar = 10^(p.x) × a₀  [incorrect, fixed]

  PARTIAL CORRELATIONS:
    r(X,Y|Z) = (r_XY - r_XZ × r_YZ) / √((1-r_XZ²)(1-r_YZ²))
    Controls for confounding between correlated galaxy properties.

  MULTIVARIATE REGRESSION:
    OLS with Gaussian elimination. No regularization.
    R², adjusted R² reported. No bootstrapped CIs (TBD).

  QUALITY CUTS:
    Applied progressively to test robustness:
    - V_max threshold (50, 80 km/s)
    - Inclination threshold (30°)
    - Number of RC points (5, 10)
    - g_bar dynamic range (0.5, 1.0 dex)
    - Combined "STRICT" and "VERY STRICT" cuts

  LIMITATIONS:
    • Subsampled data (every 2nd point) — full sample may differ slightly
    • No error bars on individual points used in fitting
    • No covariance between points within same galaxy
    • Υ_★ fixed, not marginalized (conservative choice)
    • LITTLE THINGS mostly excluded due to missing metadata
    • Quality cuts reduce sample to n=46–78; marginal SB trend
      needs confirmation with larger clean samples

SECTION 13: ERROR-WEIGHTED FIT
================================================================================

  PURPOSE: The v2.0 unweighted RMS fit treats all radial points equally.
           This is biased because:
           (a) galaxies with many points dominate the fit
           (b) face-on galaxies have large sin(i) errors
           (c) distant galaxies have larger distance uncertainties
           (d) galaxies with large internal scatter add noise

  WEIGHTING SCHEME:
    w_i = (1/sigma²_gal) * sin²(inc) * (1/sqrt(D)) * (1/sqrt(N_gal))

    where:
      sigma²_gal = within-galaxy variance of log(g_obs/g_bar)
      inc = galaxy inclination (sin²(i) correction for deprojection)
      D = distance in Mpc
      N_gal = number of points in that galaxy (downweight oversampled)

SECTION 14: HIERARCHICAL MODEL (RANDOM-EFFECTS META-ANALYSIS)
================================================================================

  PURPOSE: The largest single uncertainty in v2.0 was the
           global-vs-per-galaxy gap (±0.112 dex). This gap arises because:
           - Global fit weights all POINTS equally (oversampled galaxies dominate)
           - Per-galaxy median weights all GALAXIES equally (ignores precision)
           A hierarchical model explicitly models both levels.

  MODEL STRUCTURE:

    Level 1 (radial points):
      r_ij = log10(g_obs,ij) - log10(RAR(g_bar,ij, a₀_i)) + e_ij
      where e_ij ~ N(0, sigma²_i) is within-galaxy measurement noise

    Level 2 (galaxies):
      log(a₀_i) = mu + u_i
      where u_i ~ N(0, tau²) is between-galaxy heterogeneity

    Level 3 (population):
      mu = population-level log(a₀) — the parameter we want

  ESTIMATION METHOD: DerSimonian-Laird random-effects meta-analysis
    - Each galaxy contributes one a₀ estimate from its own McGaugh fit
    - Within-galaxy SE: se_i² = sigma²_within / n_i
    - Between-galaxy heterogeneity: tau² estimated from data
    - Final weights: w_i = 1/(se_i² + tau²) * sin²(inc) / sqrt(D)
    - I² statistic: fraction of total variance due to true heterogeneity

  v4.0 UPGRADES OVER v3.0:

  2. PER-GALAXY NUISANCE MARGINALIZATION:

       log(g_bar_adj) = log(g_bar) + log[(Y*/0.5)*(1-f_gas) + f_gas]
       log(g_obs_adj) = log(g_obs) - delta_logD
     where f_gas = 1.33*MHI / (1.33*MHI + 0.5*L36) per galaxy.

     Marginalization:
       For each grid point, fit a0 via McGaugh RAR golden section.
       Weight: exp(-0.5 * delta_chi2 / sigma2_hat) * prior(Y*) * prior(dD)
       Marginalized log(a0) = weighted mean of grid log(a0) values.
       Marginalized SE = sqrt(margVar + withinSE2).

  3. HIERARCHICAL MODEL: DerSimonian-Laird using marginalized per-galaxy
     estimates (not fixed-Y* estimates).

  ────────────────────────────────────────────────────────────────────

  EFFECT OF MARGINALIZATION:

    Mean shift in log(a0): +0.014 dex (marginalized vs fixed Y*)
    Median shift: +0.003 dex
    RMS shift: 0.221 dex
    Max |shift|: 1.040 dex

    INTERPRETATION: Marginalization has a SMALL mean effect but large
    galaxy-to-galaxy variation. Individual galaxies can shift by up to
    1 dex, but the population average barely moves.

================================================================================
  SECTION 20: SYSTEMATIC SUSPECTS TABLE
================================================================================

  A formal audit of all known factors that could bias or inflate the
  measured a₀ and its scatter. Each factor is ranked by priority,
  with a description of the mechanism, current mitigation status,
  proposed test, and elimination criterion.

  ┌─────────────────────────────────────────────────────────────────┐
  │  PRIORITY: HIGH                                                 │
  └─────────────────────────────────────────────────────────────────┘

  SUSPECT 1: Non-circular motions (bars, spiral arms, warps)
  ─────────────────────────────────────────────────────────────────
  Mechanism:    Bars and spiral arms create streaming motions that
                make the measured line-of-sight velocity deviate from
                the true circular velocity. This strikes at the heart
                of the a₀ inference because g_obs = V²/R assumes
                circular orbits.
  Bias on a₀:  Can shift individual galaxy a₀ estimates by 10-30%
                in either direction; increases tau.
  Current status:
    - SPARC quality flags (Q=1,2) exclude worst cases
    - GOLD sample removes Q=3 galaxies
    - No explicit non-circular motion correction applied
  Proposed test:
    - Compare a₀ for barred vs unbarred galaxies (from morphological
      classification)
    - Use harmonic decomposition of velocity fields (where 2D data
      exists) to quantify non-circular fraction
    - Check if residuals from RAR correlate with bar strength proxy
  Elimination criterion:
    Barred and unbarred subsamples yield a₀ consistent within 1-sigma,
    AND residuals show no correlation with bar strength.

  SUSPECT 2: Pressure support / asymmetric drift / gas disequilibrium
  ─────────────────────────────────────────────────────────────────
  Mechanism:    In dwarf and gas-rich galaxies, the gas may not be in
                clean circular equilibrium. Turbulent pressure support
                means V_circ > V_measured, making g_obs too low.
                Bursty star formation can temporarily reshape the
                potential, causing time-dependent RC distortions.
  Bias on a₀:  Systematically lowers g_obs in dwarfs, pushing their
                a₀ estimates higher and inflating tau.
  Current status:
    - No pressure support correction applied
    - GOLD+i45 sample partially mitigates by excluding low-quality
      dwarfs, but fgas>0.3 subsample still included
    - v4.0 fgas split shows fgas<0.3 gives a₀=3712 vs fgas>=0.3
      gives a₀=3500 (6% difference) — weak evidence
  Proposed test:
    - Apply asymmetric drift correction: V²_circ = V²_rot + R/rho *
      d(rho * sigma²)/dR where available
    - Compare a₀ for galaxies with known sigma_gas measurements
    - Exclude all galaxies with fgas > 0.5 and check a₀ stability
  Elimination criterion:
    a₀ shifts < 0.05 dex after asymmetric drift correction,
    AND fgas split gives < 5% difference.

  SUSPECT 3: Mass-to-light ratio Y★
  ─────────────────────────────────────────────────────────────────
  Mechanism:    The stellar contribution to g_bar depends on Y★.
                If Y★ is wrong, g_bar is wrong, shifting the RAR
                intercept and thus a₀.
  Bias on a₀:  Y★ too high -> g_bar too high -> a₀ shifts down.
                v4.0 showed +7.7% shift when marginalizing Y★.
  Current status:
    - v4.0 marginalizes over Y★ = [0.20, 0.80] with Gaussian prior
      N(0.50, 0.12²) — MITIGATED
    - Residual contribution to uncertainty: 0.020 dex
    - Key finding: marginalization does NOT reduce tau
  Proposed test:
    - Use SPS-based per-galaxy Y★ estimates (where available) instead
      of uniform prior
    - Check if a₀ changes with different prior widths (0.08 vs 0.15)
  Elimination criterion:
    Already partially eliminated. Full elimination requires
    SPS-based Y★ giving consistent a₀ within 0.03 dex.

  SUSPECT 4: Distance uncertainties
  ─────────────────────────────────────────────────────────────────
  Mechanism:    Distance errors propagate as V² ~ D and R ~ D,
                so g_obs ~ V²/R ~ D while g_bar ~ D⁻².
                This creates correlated errors along both RAR axes.
  Bias on a₀:  Can inflate scatter and bias hierarchical mean if
                distance errors are asymmetric.
  Current status:
    - v4.0 marginalizes over deltaLogD = [-0.1, +0.1] with
      N(0, 0.07²) prior — MITIGATED
    - Residual contribution: 0.015 dex
    - Distance split: D<15 gives a₀=4358 vs D>=15 gives a₀=3238
      (35% difference — LARGE, needs attention)
  Proposed test:
    - Use only galaxies with Cepheid/TRGB distances (highest quality)
    - Check if distance split shrinks with narrower prior or
      independent distance estimates
  Elimination criterion:
    Distance split reduces to < 10% difference after restricting
    to Cepheid/TRGB distances.

  SUSPECT 5: Within-galaxy covariance / weighting bias
  ─────────────────────────────────────────────────────────────────
  Mechanism:    Points within the same galaxy share systematic errors
                (distance, Y★, inclination). Treating them as
                independent inflates effective sample size and can
                bias a₀ toward galaxies with more data points.
  Bias on a₀:  Direction depends on which galaxies are oversampled.
                v3 unweighted vs hierarchical: 4837 vs 3374 (43%!).
  Current status:
    - v4.0 hierarchical model treats each galaxy as one estimate
      with its own uncertainty — MITIGATED
    - Between-galaxy heterogeneity (tau) absorbs excess scatter
    - I² = 92.4% confirms galaxies are not interchangeable
  Proposed test:
    - Compare with inverse-variance weighting at galaxy level
    - Bootstrap galaxies (not points) and check CI coverage
  Elimination criterion:
    Already mitigated by hierarchical model. Bootstrap galaxy-level
    CIs should cover nominal 68%/95% rates.

  SUSPECT 6: Inclination errors
  ─────────────────────────────────────────────────────────────────
  Mechanism:    V_rot = V_los / sin(i). At low inclination, small
                errors in i cause large errors in V_rot (and thus
                g_obs). This is particularly severe below i = 40°.
  Bias on a₀:  Low-inclination galaxies have inflated V_rot
                uncertainties, scattering a₀ estimates.
  Current status:
    - GOLD+i45 sample requires i >= 45° — MITIGATED
    - v4.0 inclination split: inc>=60 gives a₀=3995 vs inc 45-60
      gives a₀=2812 (42% difference — CONCERNING)
    - Residual uncertainty contribution: 0.021 dex
  Proposed test:
    - Restrict to i >= 60° only and check a₀ stability
    - Apply per-galaxy inclination marginalization (not done in v4.0)
  Elimination criterion:
    i>=60 subsample gives a₀ within 0.05 dex of i>=45 subsample.
    NOTE: Current data FAILS this — 42% split is a red flag.

  ┌─────────────────────────────────────────────────────────────────┐
  │  PRIORITY: MEDIUM                                               │
  └─────────────────────────────────────────────────────────────────┘

  SUSPECT 7: Gas disk flaring / disk thickness
  ─────────────────────────────────────────────────────────────────
  Mechanism:    Standard decompositions assume infinitely thin disks.
                Late-type and dwarf galaxies often have thick, flared
                gas disks whose vertical structure changes the
                gravitational potential at large radii.
  Bias on a₀:  Overestimates g_bar_gas at large R, shifting a₀
                slightly upward.
  Current status:
    - SPARC uses standard thin-disk decomposition
    - No correction applied
  Proposed test:
    - Compare thin-disk vs finite-thickness g_bar for a subset
      of galaxies with known HI scale heights
  Elimination criterion:
    Thin vs thick disk g_bar differ by < 5% at outer radii for
    the majority of the sample.

  SUSPECT 8: Beam smearing (angular resolution)
  ─────────────────────────────────────────────────────────────────
  Mechanism:    Finite beam size smooths the inner velocity gradient,
                making the RC rise more slowly. This suppresses inner
                g_obs, creating artificial "cusp-core" signatures and
                biasing the inner RAR.
  Bias on a₀:  Suppresses g_obs in the inner regions where the RAR
                transition happens, potentially shifting a₀.
  Current status:
    - SPARC compilation uses heterogeneous angular resolution
    - No explicit beam smearing correction
    - Inner points are included in the fit
  Proposed test:
    - Exclude inner 2 beam widths for each galaxy and refit
    - Compare a₀ from inner-only vs outer-only radii
  Elimination criterion:
    a₀ from R > 2*beam is within 0.05 dex of full-radius a₀.

  SUSPECT 9: Environmental / external field effects
  ─────────────────────────────────────────────────────────────────
  Mechanism:    In MOND, the external gravitational field from
                neighboring structures can break the strong equivalence
                principle and alter the effective a₀ for satellite or
                cluster galaxies (external field effect, EFE).
  Bias on a₀:  If EFE is real, galaxies in dense environments would
                show different a₀ than isolated galaxies, inflating
                population scatter.
  Current status:
    - SPARC sample is mostly field galaxies
    - No environmental classification applied
  Proposed test:
    - Classify galaxies by environmental density (field vs group
      vs cluster proximity)
    - Compare a₀ for isolated vs non-isolated subsamples
  Elimination criterion:
    Isolated and non-isolated subsamples agree within 1-sigma.

  ┌─────────────────────────────────────────────────────────────────┐
  │  PRIORITY: LOW                                                  │
  └─────────────────────────────────────────────────────────────────┘

  SUSPECT 10: Sample selection / dynamic range
  ─────────────────────────────────────────────────────────────────
  Mechanism:    Low-mass galaxies with narrow dynamic range in g_bar
                may not constrain the RAR transition at all, yielding
                unreliable a₀ estimates that inflate scatter.
  Bias on a₀:  Spurious outlier estimates from unconstrained fits.
  Current status:
    - GOLD sample requires Q<=2 — partially mitigated
    - n>=20 points vs n<20: a₀=3724 vs 3465 (7%, acceptable)
    - Vmax>150 vs 80-150: a₀=3463 vs 4725 (36%, LARGE)
  Proposed test:
    - Require minimum dynamic range in log(g_bar): at least 1 dex
    - Check if removing narrow-range galaxies reduces tau
  Elimination criterion:
    Excluding narrow-range galaxies changes a₀ by < 5% and
    reduces tau by > 0.05 dex.

  SUSPECT 11: Three-body problem / chaotic orbits
  ─────────────────────────────────────────────────────────────────
  Mechanism:    Classical three-body chaos in gravitational dynamics.
                In galaxies, orbits in barred/triaxial potentials
                can be chaotic, but RC extraction uses collective
                smooth potential, not individual orbits.
  Bias on a₀:  Negligible for rotation curves. Bars already captured
                by Suspect 1 (non-circular motions).
  Current status:
    - Not relevant for smooth-potential RC analysis
    - Bars and triaxial potentials covered under Suspect 1
  Proposed test:
    - Not required for current analysis scope
  Elimination criterion:
    N/A — this is not a primary suspect for RC-based a₀.

  The acceleration transition scale is robust to kinematic
  systematics. However, the v5.0 report's optimistic claims
  about red flag resolution were incorrect. The distance,
  inclination, and velocity splits persist and represent genuine
  systematic uncertainties in the measurement.

  The best estimate of a0, combining all corrections, is:
    a0 = 3274-3918 (km/s)^2/kpc  (range across corrections)
    = (1.06 - 1.27) x 10^-10 m/s^2
    = (0.98 - 1.22) x cH0/2pi

  This is consistent with the v4.0 headline value of 3633
  (= 1.18e-10 m/s^2 = 1.130 x cH0/2pi) and with the
  McGaugh+2016 literature value of 1.20e-10 m/s^2.

  REMAINING OPEN QUESTIONS:
    1. The distance method split (27%) is the largest remaining
       systematic. Galaxies with precise (TRGB/Cepheid/SN)
       distances give higher a0 than Hubble flow galaxies. This
       could mean: (a) Hubble flow errors bias a0 low, or (b)
       the precise-distance subsample is biased.
    2. The distance residual correlation (r = -0.31) suggests
       a distance-dependent systematic. Further investigation
       with volume-limited samples would help.
    3. The cosmological ratio a0/(cH0/2pi) ranges from 1.0 to
       1.2 depending on corrections applied — consistent with
       but not definitively equal to unity.

================================================================================
  SECTION 22: PHASE 6 — MATCHED-SAMPLE SPLIT RESOLUTION
================================================================================

  Phase 6 asks: Are the three remaining red-flag splits (distance,
  inclination, velocity) genuine a0 dependencies, or confounding
  effects from sample composition differences?

  METHOD: Nearest-neighbor matching. For each split variable, pair
  galaxies from each group that are matched on confounders (Vmax,
  inc, fgas, n_pts, distance — whichever are NOT the split variable).
  Then compare a0 between matched pairs using both hierarchical DL
  and paired t-tests. Also: delete-1 jackknife for significance.

  Script: scripts/phase6-matched-splits.cjs (v6.0.0)
  Results: public/phase6-matched-results.json
  Sample: GOLD+i45 = 59 galaxies (same as v4.0/Phase 5)

  Threshold: delta < 0.05 dex (~12%) = RESOLVED
             delta 0.05-0.10 dex = INCONCLUSIVE
             delta > 0.10 dex = UNRESOLVED

  ────────────────────────────────────────────────────────────────────
  SPLIT 1: DISTANCE METHOD (Precise vs Hubble-flow)
  ────────────────────────────────────────────────────────────────────

  1A. Raw split (unmatched):
      Precise (TRGB/Cepheid/SN):  n=17, a0=4243, tau=0.257, ratio=1.320
      Hubble flow:                n=30, a0=3331, tau=0.315, ratio=1.036
      UMa cluster:                n=9,  a0=4309, tau=0.258, ratio=1.340

      Raw delta: 0.105 dex (25%)

  1B. Sample composition check:
      Precise galaxies are CLOSER (median D=9.8 vs 39.5 Mpc),
      have LOWER Vmax (median 157 vs 223), and HIGHER inclination
      (median 74 vs 64). Gas fraction similar.

  1C. Matched analysis (17 matched pairs, controlled for Vmax/inc/fgas/n):
      Matched precise:     a0=4243, tau=0.257
      Matched Hubble-flow: a0=3184, tau=0.247

      Matched delta: 0.125 dex (29%) — INCREASED from raw
      Paired t-test: t = 1.07 (df=16) — NOT significant

      VERDICT: UNRESOLVED (delta > 0.10 dex) but NOT significant (t < 2.0)

  1D. Distance binning (continuous):
      D < 10 Mpc:    n=18, a0=4116, tau=0.273, ratio=1.280
      10-20 Mpc:     n=19, a0=4178, tau=0.253, ratio=1.299
      20-40 Mpc:     n=8,  a0=3402, tau=0.467, ratio=1.058
      D >= 40 Mpc:   n=14, a0=2411, tau=0.130, ratio=0.750

      Clear monotonic trend: a0 DECREASES with distance.
      Nearby galaxies (D<20): a0 ~ 4100-4200
      Distant galaxies (D>40): a0 ~ 2400

  INTERPRETATION: The distance split persists after matching and
  is NOT reduced by controlling for Vmax/inc/fgas composition.
  However, the paired t-test is not significant (t=1.07), meaning
  the per-galaxy scatter is too large to confirm the split at the
  individual level — the analysis is underpowered. The monotonic
  distance trend is suggestive of either (a) Hubble-flow distance
  errors systematically biasing a0 downward at large D, or (b) a
  distance-dependent effect, but neither can be confirmed here.

  ────────────────────────────────────────────────────────────────────
  SPLIT 2: INCLINATION (inc >= 60 vs 45-59)
  ────────────────────────────────────────────────────────────────────

  2A. Raw split:
      Inc >= 60:  n=43, a0=3995, tau=0.277, ratio=1.243
      Inc 45-59:  n=16, a0=2812, tau=0.301, ratio=0.875

      Raw delta: 0.153 dex (33%)

  2B. Sample composition check:
      Vmax, distance, fgas, n_pts are all well-matched between groups.
      No obvious confounders.

  2C. Matched analysis (16 matched pairs, controlled for Vmax/dist/fgas/n):
      Matched inc>=60:   a0=4176, tau=0.306
      Matched inc 45-59: a0=2812, tau=0.301

      Matched delta: 0.172 dex (38%) — INCREASED from raw
      Paired t-test: t = 1.24 (df=15) — NOT significant

      VERDICT: UNRESOLVED (delta > 0.10 dex) but NOT significant (t < 2.0)

  2D. Inclination binning (continuous):
      45-54:     n=10, a0=2271, tau=0.313, ratio=0.706
      55-64:     n=18, a0=3655, tau=0.287, ratio=1.137
      65-74:     n=15, a0=4800, tau=0.251, ratio=1.493
      inc >= 75: n=16, a0=3725, tau=0.279, ratio=1.159

      Non-monotonic pattern: a0 peaks at inc=65-74, then decreases
      at the highest inclinations (>= 75). This is consistent with
      sin(i) correction contamination: at moderate inclinations (45-55),
      sin(i) overcorrection reduces V_obs; at very high inclinations,
      other effects (line-of-sight confusion, warps) compete.

  INTERPRETATION: The inclination split is the largest single split
  in the sample (0.17 dex matched). It persists after controlling
  for mass, distance, and gas fraction. This is consistent with
  sin(i) deprojection systematics: V_circ = V_obs/sin(i) amplifies
  errors at lower inclinations. However, the paired t-test (t=1.24)
  means the effect is not individually significant — the analysis
  is underpowered. The effect-size persistence without statistical
  significance means we cannot distinguish this from scatter.

  ────────────────────────────────────────────────────────────────────
  SPLIT 3: VELOCITY / MASS (Vmax > 150 vs 80-150)
  ────────────────────────────────────────────────────────────────────

  3A. Raw split:
      Vmax > 150:   n=41, a0=3463, tau=0.304, ratio=1.077
      Vmax 80-150:  n=12, a0=4725, tau=0.141, ratio=1.469
      Vmax 50-80:   n=6,  a0=2781, tau=0.462, ratio=0.865

      Raw delta (hi vs mid): 0.135 dex (35%)

  3B. Sample composition check:
      Vmax>150 are more DISTANT (median 26 vs 16 Mpc) and have
      LOWER inclination (median 66 vs 67). Gas fraction similar.

  3C. Matched analysis (12 pairs, controlled for dist/inc/fgas/n):
      Matched Vmax>150:   a0=4101, tau=0.298
      Matched Vmax 80-150: a0=4725, tau=0.141

      Matched delta: 0.061 dex (17%) — REDUCED from 0.135
      Paired t-test: t = 0.77 (df=11) — NOT significant

      VERDICT: INCONCLUSIVE (delta 0.05-0.10 dex)

  3D. Velocity binning (continuous):
      50-80:     n=6,  a0=2781, tau=0.462
      80-120:    n=5,  a0=6721, tau=0.202
      120-180:   n=12, a0=3415, tau=0.129
      180-250:   n=22, a0=3111, tau=0.365
      Vmax>=250: n=14, a0=4319, tau=0.251

      Non-monotonic pattern. The 80-120 bin (n=5) has the highest
      a0 (6721) but very few galaxies. The highest-Vmax bin (>=250)
      recovers to 4319. No clear mass-dependent trend.

  INTERPRETATION: The velocity split is substantially REDUCED by
  matching (0.135 -> 0.061 dex), suggesting it was partly driven by
  distance and inclination confounders. After matching, the remaining
  0.061 dex is in the INCONCLUSIVE zone. The reduction is consistent
  with velocity acting as a proxy for distance (massive galaxies
  tend to be farther), though the analysis is underpowered to
  confirm this interpretation definitively.

  CONCLUSION:

  There is a real, robust acceleration transition scale in galaxy
  rotation curves. After cleaning, marginalization, hierarchical
  modeling, and matched-sample testing (Phases 1-6), no subsample
  split has achieved definitive statistical significance, despite
  some effect-size differences persisting in distance and inclination.

  Therefore it cannot currently be determined whether these
  differences are genuine systematics or fluctuations arising from
  the large between-galaxy scatter (tau = 0.29 dex).

  The v4.0 headline value a0 = 3633 (km/s)^2/kpc (= 1.18e-10 m/s^2)
  remains the most defensible single number. Its strict universality
  (independence from all sample properties) is neither confirmed
  nor refuted by the current data.

  Defensible range: a0 = 2800-4300 (km/s)^2/kpc = (0.9-1.4)e-10 m/s^2
  This range spans the observed subsample variation and is dominated
  by the between-galaxy heterogeneity (tau = 0.29 dex).

  The cosmological ratio a0/(cH0/2pi) = 0.87-1.34 across subsamples,
  with the headline value at 1.13. Consistency with unity is possible
  but not established with current statistical power.

  PAPER-READY STATEMENT
  ────────────────────────────────────────────────────────────────────

  "A robust acceleration transition scale exists in the rotation
  curves of late-type galaxies. Using 59 GOLD-quality SPARC galaxies
  (Vmax >= 50 km/s, inc >= 45 deg, n >= 5 points, g_bar dynamic range
  >= 1 dex) with per-galaxy stellar mass-to-light ratio and distance
  marginalization under a DerSimonian-Laird random-effects model, we
  measure a0 = 3633 (km/s)^2/kpc = 1.18 x 10^-10 m/s^2, with
  between-galaxy scatter tau = 0.291 dex (I^2 = 92.4%). This is
  consistent within 2% of the McGaugh et al. (2016) value of
  1.20 x 10^-10 m/s^2.

  Matched-sample analysis reveals effect-size differences associated
  with distance measurement method (0.125 dex), inclination bin
  (0.172 dex), and maximum rotation velocity (0.061 dex). However,
  none of these splits achieves statistical significance (jackknife
  t = 1.03, 1.53, 0.77 respectively; all < 2.0). Power analysis
  indicates that resolving these splits at the 5% significance level
  would require approximately 100-400 GOLD-quality galaxies,
  depending on the split. The velocity effect is substantially
  reduced by matching on distance and inclination, suggesting it
  is not an independent systematic.

  We conclude that the EXISTENCE of the transition scale is
  observationally robust, but whether its PRECISE VALUE is strictly
  universal — independent of distance, inclination, and galaxy mass
  — cannot be determined with the current sample of 59 galaxies.
  The limiting factor is statistical power, not methodology."

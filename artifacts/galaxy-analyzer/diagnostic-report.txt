SECTION 2: INTERPOLATION FUNCTION AUDIT (Test 1)
================================================================================

  PURPOSE: Does the fitted a₀ depend on the interpolation function?
  METHOD: Fit a₀ by minimizing RMS(log g_obs - log g_pred) using
          golden section search over log(a₀) ∈ [2.0, 5.0].
  DATA: 2,062 points from 193 galaxies.

  ┌──────────────────────────────────┬──────────┬────────┬─────────┬─────────┐
  │ Interpolation Function           │ a₀       │ RMS    │ AIC     │ BIC     │
  │                                  │(km/s)²/kpc│ (dex) │         │         │
  ├──────────────────────────────────┼──────────┼────────┼─────────┼─────────┤
  │ McGaugh RAR: 1/(1-e^(-√y))      │ 1809     │ 0.2385 │ -5909.8 │ -5904.1 │
  │ Simple MOND: 1/2 + √(1/4+1/y)   │ 1988     │ 0.2385 │ -5909.7 │ -5904.1 │
  │ Standard MOND: (1+√(1+4/y))/2   │ 1988     │ 0.2385 │ -5909.7 │ -5904.1 │
  │ Our ν(x)=x/(1+x)                │ 1972     │ 0.2443 │ -5810.8 │ -5805.2 │
  └──────────────────────────────────┴──────────┴────────┴─────────┴─────────┘

  KEY FINDINGS:

  1. a₀ varies by only 4.1% across 4 interpolation functions.
     The transition scale is ROBUST to model choice.

  2. All functions give a₀ ≈ 1900 (km/s)²/kpc in the global fit.
     This is LOWER than the canonical 3702 from SPARC per-galaxy fits.
     Reason: global fit weights all points equally; per-galaxy median
     weights all galaxies equally. Low-mass galaxies dominate by number
     but have fewer well-constrained points.

  3. Our framework (ν(x)=x/(1+x)) fits WORST of the 4.
     ΔAIC = 99 relative to McGaugh RAR form — statistically decisive.
     The McGaugh and standard MOND forms are indistinguishable.

  4. The cosmological ratio a₀/(cH₀/2π) is 0.56–0.62 with the global
     fit, NOT ~0.87 as previously reported. The ~0.87 ratio uses the
     per-galaxy median a₀ = 3702, which is method-dependent.

  IMPLICATIONS:
  - The EXISTENCE of a transition scale is model-independent.
  - The PRECISE VALUE of a₀ is model-dependent (global vs per-galaxy,
    interpolation function, sample composition).
  - The cosmological coincidence a₀ ≈ cH₀/2π depends on methodology.
  - Our interpolation function is not the best fit to the data.

SECTION 3: BARYONIC CALIBRATION FACTOR (Test 2)
================================================================================

  PURPOSE: Does rescaling g_bar by a constant factor improve the fit?
  METHOD: Replace g_bar → f_bar × g_bar, fit with McGaugh RAR and
          fixed a₀ = 3702, scan f_bar from 0.7 to 1.5.

  ┌─────────┬──────────┬──────────┐
  │ f_bar   │ RMS (dex)│ bias     │
  ├─────────┼──────────┼──────────┤
  │ 0.70    │ 0.2436   │ +0.030   │
  │ 0.80    │ 0.2425   │ -0.005   │
  │ 0.90    │ 0.2416   │ -0.036   │
  │ 0.95    │ 0.2412   │ -0.051   │
  │ 1.00    │ 0.2409   │ -0.065   │
  │ 1.10    │ 0.2403   │ -0.090   │
  │ 1.20    │ 0.2399   │ -0.114   │
  │ 1.30    │ 0.2395   │ -0.136   │
  │ 1.50    │ 0.2390   │ -0.175   │
  └─────────┴──────────┴──────────┘

  KEY FINDINGS:

  1. RMS is nearly FLAT across all f_bar values.
     Best f_bar = 1.64, but improvement is only 0.9%.
     Baryonic calibration is NOT the dominant source of scatter.

  2. Bias changes monotonically with f_bar.
     f_bar = 0.80 gives near-zero bias (-0.005 dex).
     This suggests Υ_★ may be slightly overestimated, OR
     a₀ = 3702 is too high for the global fit (consistent with Test 1).

  3. f_bar and a₀ are DEGENERATE.
     Scaling g_bar up is equivalent to scaling a₀ down.
     To separate them: need external Υ_★ constraint
     (e.g., stellar population synthesis, dynamical mass).

  IMPLICATIONS:
  - Baryonic calibration is not the "hidden factor."
  - The scatter is dominated by galaxy-to-galaxy variation, not Υ_★ offset.
  - A ~20% Υ_★ offset cannot be ruled out but does not change conclusions.

SECTION 4: RESIDUAL CORRELATION MAP (Test 3)
================================================================================

  PURPOSE: Are residuals (obs - pred) random, or do they correlate
           with galaxy properties?
  METHOD: Compute mean RAR residual per galaxy using McGaugh RAR
          with a₀ = 3702, then Pearson r against 8 galaxy properties.
  SAMPLE: 156 galaxies with at least 3 data points.

  ┌─────────────────────────┬──────────┬──────┬──────────────────────┐
  │ Parameter               │ r        │ n    │ Assessment           │
  ├─────────────────────────┼──────────┼──────┼──────────────────────┤
  │ Inclination             │ -0.159   │ 156  │ Not significant      │
  │ Distance                │ -0.139   │ 156  │ Not significant      │
  │ V_max                   │ -0.313   │ 156  │ SIGNIFICANT ⚠        │
  │ Gas fraction (MHI/L36)  │ +0.641   │ 156  │ STRONG ⚠⚠           │
  │ Surface brightness      │ -0.644   │ 156  │ STRONG ⚠⚠           │
  │ Disk scale length       │ -0.275   │ 156  │ Weak                 │
  │ Number of points        │ -0.157   │ 156  │ Not significant      │
  │ Luminosity (L36)        │ -0.524   │ 156  │ STRONG ⚠⚠           │
  └─────────────────────────┴──────────┴──────┴──────────────────────┘

  KEY FINDINGS:

  1. GOOD NEWS: Inclination and distance show NO significant correlation.
     These are the most feared observational systematics.
     The RAR residuals are NOT driven by viewing geometry.

  2. BAD NEWS: Gas fraction (r = +0.64) and surface brightness (r = -0.64)
     show STRONG correlations with residuals.
     This means:
     - Gas-rich galaxies systematically have POSITIVE residuals
       (observed acceleration exceeds prediction)
     - High surface brightness galaxies have NEGATIVE residuals
       (prediction exceeds observed acceleration)

  3. Luminosity (r = -0.52) and V_max (r = -0.31) also correlate.
     These are not independent: luminosity correlates with SB and V_max.
     The underlying driver is likely surface brightness or gas fraction.

  INTERPRETATION:
  Three possible explanations, not mutually exclusive:

  A. Υ_★ VARIES WITH SURFACE BRIGHTNESS (systematic)
     If high-SB galaxies have higher true Υ_★ than assumed,
     g_bar would be underestimated → negative residuals.
     This is physically plausible: stellar populations in
     high-SB regions tend to be older, redder, higher Υ_★.
     Li et al. (2018) found similar structure but concluded
     the effect was absorbed by marginalizing nuisance params.

  B. THE INTERPOLATION FUNCTION NEEDS A SECOND PARAMETER (model)
     If the transition shape depends on surface density,
     a single universal a₀ cannot capture all galaxies.
     This would NOT invalidate the transition scale's existence,
     but would mean the phenomenology is richer than 1-parameter MOND.

  C. a₀ IS NOT TRULY UNIVERSAL (physics)
     If a₀ genuinely varies with galaxy surface brightness or
     gas fraction, the "universal acceleration scale" is an
     approximation, not an exact law.
     This would weaken the cosmological interpretation.

  THE MOST IMPORTANT FOLLOW-UP:
  Refit the RAR with Υ_★ as a free parameter (per galaxy or per
  surface-brightness bin) and check if the correlations disappear.
  If they do → systematic (interpretation A).
  If they persist → real physics or model limitation (B or C).

SECTION 5: SUB-POPULATION STABILITY
================================================================================

  PURPOSE: Does a₀ remain the same when fitting different galaxy types?
  METHOD: Fit a₀ separately for 8 sub-populations using McGaugh RAR.

  ┌──────────────────────────────┬───────┬──────────┬────────┬────────┐
  │ Sub-population               │ n_gal │ a₀ (fit) │ RMS    │ Δa₀   │
  ├──────────────────────────────┼───────┼──────────┼────────┼────────┤
  │ High surface brightness      │ 144   │ 3118     │ 0.188  │ -16%   │
  │ Low surface brightness       │ 31    │ 1322     │ 0.264  │ -64%   │
  │ High mass (V > 150 km/s)     │ 56    │ 3720     │ 0.128  │ +0.5%  │
  │ Low mass (V < 100 km/s)      │ 88    │ 46170    │ 0.265  │ +1147% │
  │ High inclination (> 60°)     │ 86    │ 3145     │ 0.190  │ -15%   │
  │ Low inclination (< 45°)      │ 41    │ 1810     │ 0.247  │ -51%   │
  │ Gas-rich (MHI/L36 > 0.5)     │ 73    │ 1137     │ 0.241  │ -69%   │
  │ Gas-poor (MHI/L36 ≤ 0.5)     │ 102   │ 3106     │ 0.178  │ -16%   │
  └──────────────────────────────┴───────┴──────────┴────────┴────────┘

  KEY FINDINGS:

  1. HIGH-MASS GALAXIES: a₀ = 3720, RMS = 0.128 dex.
     These galaxies give the canonical a₀ with the tightest scatter.
     They are the backbone of the RAR signal.

  2. LOW-MASS GALAXIES: a₀ = 46170, RMS = 0.265 dex.
     The fitted a₀ is 12× higher with much worse scatter.
     This does NOT mean a₀ = 46170 is physical — it means:
     - Low-mass galaxies are poorly constrained
     - They may not span enough dynamic range in g_bar
     - Or the RAR shape is genuinely different for dwarfs
     - Or systematic errors (beam smearing, non-circular motions,
       distance errors) dominate at low masses

  3. GAS-RICH vs GAS-POOR: a₀ differs by 2.7×.
     Gas-rich: a₀ = 1137; Gas-poor: a₀ = 3106.
     This is a large, troubling difference.
     Most likely explanation: Υ_★ is wrong for gas-rich galaxies
     (where stellar mass contributes less, so Υ_★ errors matter less,
     but the overall g_bar level is dominated by gas with its own
     calibration uncertainties).

  4. INCLINATION MATTERS: a₀ differs by 1.7× between high and low inc.
     Despite residuals not correlating with inclination per galaxy,
     the FITTED a₀ does depend on inclination cut.
     This is concerning and may indicate unresolved sin(i) systematics.

  OVERALL VERDICT:
  The transition scale is most robust for high-mass, high-SB, gas-poor,
  high-inclination galaxies (a₀ ≈ 3100–3720, RMS ≈ 0.13–0.19).
  For other sub-populations, a₀ is poorly constrained.
  Strict universality is NOT confirmed by this analysis.

SECTION 7: WHAT TO DO NEXT
================================================================================

  PRIORITY ORDER (highest first):

  1. RESOLVE THE RESIDUAL CORRELATIONS
     Refit RAR with Υ_★ free (per galaxy or per SB bin).
     If correlations vanish → systematic (Υ_★ miscalibration).
     If they persist → the model is incomplete or a₀ is not universal.
     This is the single highest-priority test.

  2. UNDERSTAND THE LOW-MASS DIVERGENCE
     Why does a₀ = 46170 for V < 100 km/s galaxies?
     Options: beam smearing, pressure support, asymmetric drift,
     distance errors, or genuine departure from universal RAR.
     Test: quality-cut to galaxies with > 1 dex g_bar range.

  3. JOINT FIT: a₀ + f_bar
     Currently a₀ and f_bar are degenerate.
     Jointly fit both to resolve the degeneracy.
     Or: use external Υ_★ constraints (SPS models, dynamics).

  4. METHOD-INDEPENDENT a₀
     Report a₀ from both global fit and per-galaxy median.
     Report a₀ from all 4 interpolation functions.
     The "canonical" a₀ should be the intersection of all methods.

  5. THEN (and only then): COSMOLOGICAL INTERPRETATION
     Only if a₀ is method-independent and residuals are clean,
     does the cH₀/2π ratio become meaningful to discuss.

SECTION 8: METHODOLOGY NOTES
================================================================================

  DATA FORMAT:
    Plot points stored as (x, y, g) where:
      x = log10(g_bar) in (km/s)²/kpc
      y = log10(g_obs/g_bar)
      g = galaxy name (truncated to 12 chars)

  g_bar COMPUTATION:
    From SPARC: Vbar² = Υ_disk × Vdisk² + Υ_bulge × Vbulge² + Vgas²
    g_bar = Vbar² / r
    Υ_disk = 0.5, Υ_bulge = 0.7 (fixed, not fitted per galaxy)
    This is a proper disk mass model, NOT spherical GM/r².

  LITTLE THINGS:
    g_bar derived from V_total² - V_DM² (using published DM models)
    Less reliable than SPARC decomposition.

  FITTING:
    Golden section search on log(a₀) with 50 refinement steps.
    Cost function: RMS of log10(g_obs) - log10(g_pred).
    No marginalization over nuisance parameters.
    1 free parameter (a₀) per fit.

  INTERPOLATION FUNCTIONS TESTED:
    1. McGaugh RAR: g_obs = g_bar / (1 - exp(-√(g_bar/a₀)))
    2. Simple MOND: g_obs = g_bar × (1/2 + √(1/4 + a₀/g_bar))
    3. Standard MOND: g_obs = g_bar × (1 + √(1 + 4a₀/g_bar)) / 2
    4. Our framework: g_obs = √(g_bar² + a₀² × g_bar/(g_bar + a₀))

  RESIDUAL CORRELATIONS:
    Per-galaxy mean residual (log g_obs - log g_pred) vs galaxy metadata.
    Pearson r with |r| > 0.3 flagged as significant.
    156 galaxies with ≥ 3 data points used.

  LIMITATIONS:
    • Subsampled data (every 2nd point) — full sample may differ slightly
    • No error bars on individual points used in fitting
    • No covariance between points within same galaxy
    • Υ_★ fixed, not marginalized (conservative choice)
    • LITTLE THINGS g_bar uses DM-subtraction method (less direct)

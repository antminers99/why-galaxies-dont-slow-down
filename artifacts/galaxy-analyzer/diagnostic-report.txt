SECTION 11: METHODOLOGY NOTES
================================================================================

  DATA FORMAT:
    Plot points stored as (x, y, g) where:
      x = log10(g_bar) in (km/s)ВІ/kpc
      y = log10(g_obs/g_bar)
      g = galaxy name (truncated to 12 chars)

  g_bar COMPUTATION:
    From SPARC: VbarВІ = ОҘ_disk Г— VdiskВІ + ОҘ_bulge Г— VbulgeВІ + VgasВІ
    g_bar = VbarВІ / r
    ОҘ_disk = 0.5, ОҘ_bulge = 0.7 (fixed, not fitted per galaxy)
    This is a proper disk mass model, NOT spherical GM/rВІ.

  LITTLE THINGS:
    g_bar derived from V_totalВІ - V_DMВІ (using published DM models)
    Less reliable than SPARC decomposition.
    Only 2 LITTLE THINGS galaxies in analysis subset (most lack metadata).

  FITTING:
    Golden section search on log(aвӮҖ) with 50 refinement steps.
    Cost function: RMS of log10(g_obs) - log10(g_pred).
    No marginalization over nuisance parameters.
    1 free parameter (aвӮҖ) per fit.

  INTERPOLATION FUNCTIONS TESTED:
    1. McGaugh RAR: g_obs = g_bar / (1 - exp(-вҲҡ(g_bar/aвӮҖ)))
    2. Simple MOND: g_obs = g_bar Г— (1/2 + вҲҡ(1/4 + aвӮҖ/g_bar))
    3. Standard MOND: g_obs = g_bar Г— (1 + вҲҡ(1 + 4aвӮҖ/g_bar)) / 2
    4. Our framework: g_obs = вҲҡ(g_barВІ + aвӮҖВІ Г— g_bar/(g_bar + aвӮҖ))

  RESIDUAL COMPUTATION (corrected in Phase 2):
    Per-galaxy mean residual:
      gbar = 10^(p.x)                         [correct]
      gobs = gbar Г— 10^(p.y)
      pred = mcgaughRAR(gbar, aвӮҖ)
      resid = log10(gobs) - log10(pred)
    Phase 1 BUG: used gbar = 10^(p.x) Г— aвӮҖ  [incorrect, fixed]

  PARTIAL CORRELATIONS:
    r(X,Y|Z) = (r_XY - r_XZ Г— r_YZ) / вҲҡ((1-r_XZВІ)(1-r_YZВІ))
    Controls for confounding between correlated galaxy properties.

  MULTIVARIATE REGRESSION:
    OLS with Gaussian elimination. No regularization.
    RВІ, adjusted RВІ reported. No bootstrapped CIs (TBD).

  QUALITY CUTS:
    Applied progressively to test robustness:
    - V_max threshold (50, 80 km/s)
    - Inclination threshold (30В°)
    - Number of RC points (5, 10)
    - g_bar dynamic range (0.5, 1.0 dex)
    - Combined "STRICT" and "VERY STRICT" cuts

  LIMITATIONS:
    вҖў Subsampled data (every 2nd point) вҖ” full sample may differ slightly
    вҖў No error bars on individual points used in fitting
    вҖў No covariance between points within same galaxy
    вҖў ОҘ_вҳ… fixed, not marginalized (conservative choice)
    вҖў LITTLE THINGS mostly excluded due to missing metadata
    вҖў Quality cuts reduce sample to n=46вҖ“78; marginal SB trend
      needs confirmation with larger clean samples

SECTION 13: ERROR-WEIGHTED FIT
================================================================================

  PURPOSE: The v2.0 unweighted RMS fit treats all radial points equally.
           This is biased because:
           (a) galaxies with many points dominate the fit
           (b) face-on galaxies have large sin(i) errors
           (c) distant galaxies have larger distance uncertainties
           (d) galaxies with large internal scatter add noise

  WEIGHTING SCHEME:
    w_i = (1/sigmaВІ_gal) * sinВІ(inc) * (1/sqrt(D)) * (1/sqrt(N_gal))

    where:
      sigmaВІ_gal = within-galaxy variance of log(g_obs/g_bar)
      inc = galaxy inclination (sinВІ(i) correction for deprojection)
      D = distance in Mpc
      N_gal = number of points in that galaxy (downweight oversampled)

  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ

  в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ

SECTION 14: HIERARCHICAL MODEL (RANDOM-EFFECTS META-ANALYSIS)
================================================================================

  PURPOSE: The largest single uncertainty in v2.0 was the
           global-vs-per-galaxy gap (Вұ0.112 dex). This gap arises because:
           - Global fit weights all POINTS equally (oversampled galaxies dominate)
           - Per-galaxy median weights all GALAXIES equally (ignores precision)
           A hierarchical model explicitly models both levels.

  MODEL STRUCTURE:

    Level 1 (radial points):
      r_ij = log10(g_obs,ij) - log10(RAR(g_bar,ij, aвӮҖ_i)) + e_ij
      where e_ij ~ N(0, sigmaВІ_i) is within-galaxy measurement noise

    Level 2 (galaxies):
      log(aвӮҖ_i) = mu + u_i
      where u_i ~ N(0, tauВІ) is between-galaxy heterogeneity

    Level 3 (population):
      mu = population-level log(aвӮҖ) вҖ” the parameter we want

  ESTIMATION METHOD: DerSimonian-Laird random-effects meta-analysis
    - Each galaxy contributes one aвӮҖ estimate from its own McGaugh fit
    - Within-galaxy SE: se_iВІ = sigmaВІ_within / n_i
    - Between-galaxy heterogeneity: tauВІ estimated from data
    - Final weights: w_i = 1/(se_iВІ + tauВІ) * sinВІ(inc) / sqrt(D)
    - IВІ statistic: fraction of total variance due to true heterogeneity

  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ

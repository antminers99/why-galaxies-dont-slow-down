PART I: PHYSICAL SETUP AND EQUATIONS
================================================================================

────────────────────────────────────────────────────────────────────
1. THE RADIAL ACCELERATION RELATION (RAR)
────────────────────────────────────────────────────────────────────

The RAR relates the observed centripetal acceleration g_obs to the
baryonic (Newtonian) acceleration g_bar at each radius in a galaxy:

  g_obs = V_rot^2 / r          [observed, from rotation curve]
  g_bar = V_bar^2 / r          [predicted from visible mass]

The McGaugh RAR interpolation function (our sole estimator):

  g_obs = g_bar / (1 - exp(-sqrt(g_bar / a0)))

where a0 is the acceleration scale parameter.

Equivalent log-space form (used in fitting):

  log10(g_obs) = log10(g_bar) - log10(1 - exp(-sqrt(g_bar / a0)))

Three other interpolation functions were tested (Section 2 audit):

  Simple MOND:    g_obs = g_bar * (1/2 + sqrt(1/4 + a0/g_bar))
  Standard MOND:  g_obs = g_bar * (1 + sqrt(1 + 4*a0/g_bar)) / 2
  Our framework:  g_obs = sqrt(g_bar^2 + a0^2 * g_bar/(g_bar + a0))

All give a0 within 4.1% — the transition scale is model-independent.
Our framework fits WORST (dAIC=99 vs McGaugh). McGaugh is the standard.

────────────────────────────────────────────────────────────────────
2. BARYONIC ACCELERATION COMPUTATION
────────────────────────────────────────────────────────────────────

From SPARC data files, at each radius r:

  V_bar^2 = Y_disk * V_disk^2 + Y_bulge * V_bulge^2 + V_gas^2
  g_bar = V_bar^2 / r

where:
  V_disk  = disk rotation velocity contribution (from 3.6um photometry)
  V_bulge = bulge rotation velocity contribution
  V_gas   = gas rotation velocity contribution (from HI 21cm)
  Y_disk  = stellar mass-to-light ratio for disk (M_sun / L_sun)
  Y_bulge = stellar mass-to-light ratio for bulge

Fixed values (v3.0): Y_disk = 0.5, Y_bulge = 0.7
Marginalized (v4.0): Y_disk scanned over [0.20, 0.80] with 13 grid values
  Prior: Gaussian N(0.50, 0.12^2) centered on population synthesis value

Units: g_bar and g_obs in (km/s)^2/kpc
Conversion: 1 (km/s)^2/kpc = 3.241e-14 m/s^2

────────────────────────────────────────────────────────────────────
3. OBSERVED ACCELERATION
────────────────────────────────────────────────────────────────────

From rotation curves:

  V_rot = V_los / sin(i)    [deprojection]
  g_obs = V_rot^2 / r

where:
  V_los = line-of-sight velocity from HI or H-alpha spectroscopy
  i     = galaxy inclination (degrees)
  r     = galactocentric radius (kpc)

Inclination correction is critical:
  sin(i) errors amplify at low i (< 45 deg)
  This is why GOLD+i45 requires inc >= 45 deg

────────────────────────────────────────────────────────────────────
4. UNIT CONVERSIONS
────────────────────────────────────────────────────────────────────

  a0 in (km/s)^2/kpc --> m/s^2:
    a0_ms2 = a0_kpc * 3.241e-14

  Literature canonical: a0 = 1.20e-10 m/s^2 = 3703 (km/s)^2/kpc
  Our v4.0 result:      a0 = 1.18e-10 m/s^2 = 3633 (km/s)^2/kpc

  Cosmological constants:
    c     = 299,792,458 m/s
    H0    = 67.4 km/s/Mpc = 2.184e-18 s^-1
    c*H0  = 6.548e-10 m/s^2
    c*H0/(2*pi) = 1.042e-10 m/s^2

  Ratio: a0 / (c*H0/(2*pi)):
    v4.0: 1.18e-10 / 1.042e-10 = 1.130
    v3.0: 1.09e-10 / 1.042e-10 = 1.049
    Literature: 1.20e-10 / 1.042e-10 = 1.15

PART II: FITTING METHODOLOGY
================================================================================

────────────────────────────────────────────────────────────────────
5. PER-GALAXY a0 FITTING
────────────────────────────────────────────────────────────────────

For each galaxy i with n_i radial points:

  Cost function:
    chi^2_i(a0) = sum_{j=1}^{n_i} [log10(g_obs,ij) - log10(RAR(g_bar,ij, a0))]^2

  RMS:
    RMS_i(a0) = sqrt(chi^2_i / n_i)

  Optimization: Golden section search on log10(a0) in [2.0, 5.0]
    100 refinement steps, convergence to ~10^-5 dex

  Output per galaxy:
    a0_i     = best-fit acceleration scale
    RMS_i    = residual scatter around best fit
    sigma_i  = within-galaxy standard deviation of residuals

────────────────────────────────────────────────────────────────────
6. v4.0 NUISANCE MARGINALIZATION
────────────────────────────────────────────────────────────────────

For each galaxy, a grid of 65 evaluations: 13 Y* x 5 delta_D

  Y* grid (13 values):
    [0.20, 0.25, 0.30, 0.35, 0.40, 0.45, 0.50, 0.55, 0.60, 0.65, 0.70, 0.75, 0.80]
    Prior: w(Y*) = exp(-0.5 * ((Y* - 0.50) / 0.12)^2)

  Distance marginalization (5 values):
    delta_logD = [-0.10, -0.05, 0.00, +0.05, +0.10]
    Prior: w(dD) = exp(-0.5 * (dD / 0.07)^2)

  At each grid point (Y*, dD):
    g_bar_new = Y* * V_disk^2 + 0.7 * V_bulge^2 + V_gas^2
    r_new     = r * 10^(dD)
    g_bar_adj = g_bar_new / r_new

  Marginalized a0 for galaxy i:
    log(a0_i)_marg = sum_k w_k * log(a0_ik) / sum_k w_k

  where k runs over 65 grid points and w_k = w(Y*_k) * w(dD_k)

  Effect: Y* marginalization shifts a0 by +3% to +11% depending on sample.
  GOLD+i45: 3261 (fixed) --> 3633 (marginalized), +11.4%
  But tau barely changes (+0.048 dex): heterogeneity is NOT Y*-driven.

────────────────────────────────────────────────────────────────────
7. ERROR-WEIGHTED FITTING (v3.0)
────────────────────────────────────────────────────────────────────

Weight per data point:

  w_ij = (1/sigma^2_i) * sin^2(inc_i) * (1/sqrt(D_i)) * (1/sqrt(N_i))

where:
  sigma^2_i = within-galaxy variance of log(g_obs/g_bar)
  inc_i     = inclination in degrees
  D_i       = distance in Mpc
  N_i       = number of points in galaxy i

Weighted cost function:
  chi^2_w = sum_{i,j} w_ij * [log10(g_obs,ij) - log10(RAR(g_bar,ij, a0))]^2

Effect on GOLD+i45: a0 drops from 4837 (unweighted) to 3545 (weighted)
The -26.7% shift means unweighted was pulled up by oversampled galaxies.

PART III: HIERARCHICAL MODEL

────────────────────────────────────────────────────────────────────
8. DerSIMONIAN-LAIRD RANDOM-EFFECTS META-ANALYSIS
────────────────────────────────────────────────────────────────────

Model structure (3 levels):

  Level 1 (points within galaxy i):
    r_ij = log10(g_obs,ij) - log10(RAR(g_bar,ij, a0_i)) + e_ij
    e_ij ~ N(0, sigma^2_i)

  Level 2 (galaxies):
    log(a0_i) = mu + u_i
    u_i ~ N(0, tau^2)

  Level 3 (population):
    mu = population mean of log(a0)

Step 1: Estimate per-galaxy SE
  se_i = sigma_i / sqrt(n_i)
  where sigma_i is within-galaxy residual SD

Step 2: Estimate between-galaxy heterogeneity tau^2

  Q = sum_i w_i * (log(a0_i) - mu_FE)^2
  where w_i = 1/se_i^2 and mu_FE = sum(w_i * log(a0_i)) / sum(w_i)

  tau^2 = max(0, (Q - (k-1)) / (sum(w_i) - sum(w_i^2)/sum(w_i)))
  where k = number of galaxies

Step 3: Random-effects weights
  W_i = 1 / (se_i^2 + tau^2)

  Modified weights (incorporate observational quality):
  W_i_mod = W_i * sin^2(inc_i) / sqrt(D_i)

Step 4: Population estimate
  mu = sum(W_i_mod * log(a0_i)) / sum(W_i_mod)
  SE(mu) = 1 / sqrt(sum(W_i_mod))

Step 5: I^2 statistic
  I^2 = (Q - (k-1)) / Q * 100%
  Interpretation: fraction of total variance due to TRUE heterogeneity

────────────────────────────────────────────────────────────────────
9. CONFIDENCE INTERVALS
────────────────────────────────────────────────────────────────────

  log(a0) = mu +/- z_alpha * sqrt(SE^2 + tau^2)

  68% CI (z = 1.0):
    [10^(mu - sqrt(SE^2 + tau^2)), 10^(mu + sqrt(SE^2 + tau^2))]

  95% CI (z = 1.96):
    [10^(mu - 1.96*sqrt(SE^2 + tau^2)), 10^(mu + 1.96*sqrt(SE^2 + tau^2))]

  v4.0 GOLD+i45:
    mu    = 3.560 dex
    SE    = 0.041 dex
    tau   = 0.291 dex
    68% CI: [10^(3.560 - 0.294), 10^(3.560 + 0.294)] = [1846, 7149]
    95% CI: [10^(3.560 - 0.576), 10^(3.560 + 0.576)] = [938, 14067]

────────────────────────────────────────────────────────────────────
16. RESIDUAL COMPUTATION
────────────────────────────────────────────────────────────────────

  Per point:
    g_bar = 10^(x_j)                    [x_j from data]
    g_obs = g_bar * 10^(y_j)            [y_j from data]
    g_pred = McGaughRAR(g_bar, a0)
    resid_j = log10(g_obs) - log10(g_pred)

  Per galaxy mean residual:
    R_i = (1/n_i) * sum_j resid_ij

  Phase 1 BUG (corrected in Phase 2):
    Used g_bar = 10^(x_j) * a0 instead of 10^(x_j)
    This produced ARTIFICIAL correlations (gas r=+0.64, SB r=-0.64)

35. ALL EQUATIONS IN ONE PLACE
────────────────────────────────────────────────────────────────────

  RAR (McGaugh):
    g_obs = g_bar / (1 - exp(-sqrt(g_bar / a0)))

  Baryonic acceleration:
    g_bar = (Y_disk * V_disk^2 + Y_bulge * V_bulge^2 + V_gas^2) / r

  Observed acceleration:
    g_obs = V_rot^2 / r = (V_los / sin(i))^2 / r

  Fitting cost:
    chi^2(a0) = sum_j [log10(g_obs_j) - log10(RAR(g_bar_j, a0))]^2

  Point weights:
    w_ij = sin^2(inc_i) / (sigma_i^2 * sqrt(D_i * N_i))

  DL tau^2:
    Q = sum w_i * (theta_i - theta_FE)^2
    tau^2 = max(0, (Q-(k-1)) / (S1 - S2/S1))
    where S1=sum(w_i), S2=sum(w_i^2)

  DL weights:
    W_i = 1 / (se_i^2 + tau^2)

  Population mean:
    mu = sum(W_i * theta_i) / sum(W_i)
    SE = 1 / sqrt(sum(W_i))

  I^2 = max(0, (Q-(k-1))/Q) * 100%

  Partial correlation:
    r(X,Y|Z) = (r_XY - r_XZ*r_YZ) / sqrt((1-r_XZ^2)*(1-r_YZ^2))

  ANOVA decomposition:
    total^2 = within^2 + between^2

  Pressure correction:
    g_obs_corr = g_obs + sigma_HI^2 / h_gas

  Unit conversion:
    a0_ms2 = a0_kpc * 3.241e-14

  Cosmological ratio:
    R = a0 / (c * H0 / (2*pi))

  Phase 10 model:
    logA0_i = mu + beta * (X_i - mean(X)) / sd(X)
    dAIC = n*ln(RSS1/n) + 2*2 - (n*ln(RSS0/n) + 2*1)

  Power analysis:
    n_needed = n_current * (t_target / t_observed)^2

36. PAPER-READY STATEMENT (v10.0, incorporating Phases 1-10)
────────────────────────────────────────────────────────────────────

  "A robust acceleration transition scale exists in the rotation
  curves of late-type galaxies. Using 59 GOLD-quality SPARC galaxies
  with per-galaxy nuisance marginalization under a DerSimonian-Laird
  random-effects model, we measure a0 = 3633 (km/s)^2/kpc =
  1.18 x 10^-10 m/s^2, with between-galaxy scatter tau = 0.291 dex
  (I^2 = 92.4%), consistent within 2% of McGaugh et al. (2016).

  Four distinct scatter measures must be distinguished:
  (1) Total point RMS with global a0: 0.155 dex (GOLD+i45);
  (2) Within-galaxy point RMS (per-galaxy a0): 0.122 dex,
      matching McGaugh et al. (2016) at 0.13 dex;
  (3) Between-galaxy component of point variance: 0.096 dex
      (ANOVA residual);
  (4) SD of per-galaxy best-fit logA0: 0.275 dex
      (galaxy-level estimator spread; tau = 0.291 via DL).

  A systematic search over 10 candidate second parameters finds
  that rotation curve shape (eta_rot) is the strongest predictor
  of per-galaxy a0 (r = 0.334, p = 0.010, dAIC = -4.6), followed
  by Hubble type (r = 0.278, p = 0.043) and outer slope
  (r = -0.418, p = 0.045, n = 23). However, even the best candidate
  reduces tau by only 5.7%. Surface brightness (EFE proxy),
  inclination, and distance method have NEGLIGIBLE effects.

  We conclude:
  (1) The existence of a transition scale is robust;
  (2) The headline a0 = 3633 is not demonstrably biased;
  (3) Between-galaxy heterogeneity (tau ~ 0.29 dex) is intrinsic
      and largely unexplained by measured properties;
  (4) There is no tension with the literature — different analyses
      measure different scatter components;
  (5) Rotation curve shape (eta_rot) is a suggestive but modest
      second parameter; the bulk of heterogeneity requires
      unmeasured variables or is genuinely intrinsic;
  (6) Strict universality of a0 is disfavored but the alternative
      'a0 universal + unmeasured second parameter' remains viable."

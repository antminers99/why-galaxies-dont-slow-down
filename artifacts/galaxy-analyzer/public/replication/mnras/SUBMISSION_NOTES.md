# MNRAS Submission Notes

## Paper
- `paper.tex` — Main manuscript (MNRAS style, ~12 pages)
- `references.bib` — BibTeX bibliography

## Compilation
```bash
pdflatex paper
bibtex paper
pdflatex paper
pdflatex paper
```

## Required: MNRAS class file
Download `mnras.cls` and `mnras.bst` from:
https://www.ctan.org/pkg/mnras

## Data availability
All scripts and results: Zenodo DOI 10.5281/zenodo.19444465
SPARC: http://astroweb.cwru.edu/SPARC/
THINGS: https://www.mpia.de/THINGS/Data.html

## Submission checklist
- [ ] Verify all equations render correctly
- [ ] Add author affiliations when ready for real submission
- [ ] Prepare response to reviewers template
- [ ] Check MNRAS word/page limits
- [ ] Add figure files if generating plots
